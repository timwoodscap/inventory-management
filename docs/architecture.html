Architectural Notes

Observations for maintainers — dangling / inconsistent paths surfaced during review:
  • Tasks API — api.js calls /api/tasks* (GET/POST/DELETE/PATCH), but no such endpoints exist server-side; App.vue fails silently on mount.
  • Purchase Orders — client Create-PO flow & PurchaseOrderModal reference endpoints and a component that don't exist; purchase_orders.json is empty and its Pydantic models are unused.
  • Reports.vue is the lone outlier — Options API, direct axios (bypasses api.js), no i18n, debug logging.
  • Backlog.vue exists but is not registered in the router (superseded by the Dashboard shortages table).
  • Currency — USD→JPY uses a hardcoded rate of 150 in utils/currency.js.
